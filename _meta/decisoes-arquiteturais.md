# Decisões arquiteturais

> Formato ADR-leve. Cada decisão é uma seção com: **Contexto**, **Alternativas consideradas**,
> **Decisão**, **Consequências**. Decisões revisadas no futuro são marcadas como
> *superseded by [link]*, mas o registro original permanece.
